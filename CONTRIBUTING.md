# Contributing to Talons Skills Hub

## Fast path
1. Fork + branch.
2. Add or update skills in `skills/<slug>/SKILL.md`.
3. Run `npm run validate` and `npm run generate`.
4. Open PR with rationale and test output.

## Standards
- Keep skills concrete, reusable, and tool-agnostic.
- Preserve frontmatter schema and required sections.
- Use deterministic script output.

## Dev commands
- `npm run validate`
- `npm run generate`
- `npm run build:all`
