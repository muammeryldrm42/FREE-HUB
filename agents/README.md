# Agents

This folder contains assistant-specific starter instruction files so the repo can be used in an Open Claw-like workflow out of the box.

## Included profiles
- `agents/codex-cli/AGENTS.md`
- `agents/claude-code/CLAUDE.md`
- `agents/cursor/.cursorrules`
- `agents/gemini-cli/GEMINI.md`

These templates are intentionally lightweight and point assistants to the shared `skills/` library for reusable execution patterns.
